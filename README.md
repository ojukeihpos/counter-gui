# HakureiCounter
Extremely lightweight and portable C# program that loads and writes to text files. All it does is increment/decrement an integer. Made it to be used in combination with broadcasting software.

The background and text colour can be changed for chroma-keying purposes. I'll probably add more things, but it does all that it is currently intended to do.
